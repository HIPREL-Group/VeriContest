# Maximum Height of a Triangle

You are given two integers `red` and `blue`, the number of red and blue balls.

Build a triangle with row sizes `1, 2, 3, ...` where:

- all balls in a row have the same color
- adjacent rows have different colors

Return the maximum possible height.

## Example 1

> **Input:** `red = 2`, `blue = 4`  
> **Output:** `3`

## Example 2

> **Input:** `red = 2`, `blue = 1`  
> **Output:** `2`

## Example 3

> **Input:** `red = 1`, `blue = 1`  
> **Output:** `1`

## Example 4

> **Input:** `red = 10`, `blue = 1`  
> **Output:** `2`

## Constraints

- `1 <= red, blue <= 100`
