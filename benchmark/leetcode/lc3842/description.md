# Toggle Light Bulbs

You are given an integer array `bulbs`, where each value is between 1 and 100.

There are 100 light bulbs numbered from 1 to 100, and all bulbs start in the off state.

For each value `bulbs[i]`:

- If bulb `bulbs[i]` is currently off, switch it on.
- Otherwise, switch it off.

Return an array containing the bulb numbers that are on after all toggles, sorted in ascending order.
If no bulb is on, return an empty array.

## Example 1:

> **Input:** bulbs = [10,30,20,10]
> **Output:** [20,30]

## Example 2:

> **Input:** bulbs = [100,100]
> **Output:** []

## Constraints:

- `1 <= bulbs.length <= 100`
- `1 <= bulbs[i] <= 100`
