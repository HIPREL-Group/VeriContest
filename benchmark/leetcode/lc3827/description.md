# Count Monobit Integers

You are given an integer `n`.

An integer is called **Monobit** if all bits in its binary representation are the same.

Return the count of **Monobit** integers in the range `[0, n]` (inclusive).

## Example 1:

> **Input:** n = 1
> **Output:** 2
> **Explanation:** The integers in `[0, 1]` are `0` and `1`, and both have identical bits in binary.

## Example 2:

> **Input:** n = 4
> **Output:** 3
> **Explanation:** The integers in `[0, 4]` are `0`, `1`, `2`, `3`, `4`. Only `0`, `1`, and `3` are Monobit.

## Constraints:

- `0 <= n <= 1000`
