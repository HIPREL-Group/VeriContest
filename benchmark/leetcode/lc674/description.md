# Longest Continuous Increasing Subsequence

Given an unsorted array of integers `nums`, return the length of the longest continuous strictly increasing subsequence (i.e., subarray).

The subsequence must be strictly increasing. A subsequence of one element is considered strictly increasing.

## Example 1:

> **Input:** nums = [1,3,5,4,7]
> **Output:** 3
> **Explanation:** The longest continuous strictly increasing subsequence is [1,3,5] with length 3. Note that [1,3,5,7] is not valid because it is not continuous.

## Example 2:

> **Input:** nums = [2,2,2,2,2]
> **Output:** 1
> **Explanation:** The longest continuous strictly increasing subsequence is [2] with length 1.

## Constraints:

- $1 \leq \text{nums.length} \leq 10^4$
- $-10^9 \leq \text{nums}[i] \leq 10^9$
