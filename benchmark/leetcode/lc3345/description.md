# Smallest Divisible Digit Product I

## Example 1:

> **Input:** n = 10, t = 2
> **Output:** 10

## Example 2:

> **Input:** n = 15, t = 3
> **Output:** 16

## Constraints:

- `1 <= n <= 100`
- `1 <= t <= 10`
