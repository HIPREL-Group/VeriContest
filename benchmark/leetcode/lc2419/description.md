# Longest Subarray With Maximum Bitwise AND

You are given an integer array `nums` of length `n`.

Let $k$ be the maximum bitwise AND value of any non-empty subarray of `nums`. Return the length of the longest subarray whose bitwise AND is exactly $k$.

A subarray is a contiguous sequence of elements within an array.

## Example 1:

> **Input:** nums = [1, 2, 3, 3, 2, 2]
> **Output:** 2
> **Explanation:** The maximum possible bitwise AND is 3. The longest subarray with AND equal to 3 is [3, 3], so the answer is 2.

## Example 2:

> **Input:** nums = [1, 2, 3, 4]
> **Output:** 1
> **Explanation:** The maximum possible bitwise AND is 4. The longest subarray with AND equal to 4 is [4], so the answer is 1.

## Constraints:

- $1 \leq \texttt{nums.length} \leq 10^5$
- $1 \leq \texttt{nums}[i] \leq 10^6$
